A library for the Stack Overflow Trilogy (of web sites) API. See http://stackoverflow.com/ and http://soapidotnet.googlecode.com/.


Classes

  Class Description
SOAnswer Represents an answer.
SOFavorite Represents a favorite question.
SOPost Represents a post on Stack Overflow (question, answer, or comment).
SOQuestion Represents a question.
SORepChange Represents a change in reputation for a user.
SOUserAnswers Represents a user's answers (on one page).
SOUserFlair Represents a user's flair (stats).
SOUserQuestions Represents a user's questions (on one page).
StackOverflow The main class that handles the Stack Overflow API. This is a static class (Module in Visual Basic)static classModule and so cannot be inherited or instantiated.

Enumerations

  Enumeration Description
SortRule An enumeration representing sort rules of pages.
TrilogySite An enumeration representing different Stack Overflow sites.
